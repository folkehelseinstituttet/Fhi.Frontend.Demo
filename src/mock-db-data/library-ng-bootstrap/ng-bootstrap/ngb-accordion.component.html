 Code Documentation

Options

To open the first item on load, add activeIds="ngb-panel-0". Use comma separation to open multiple items.

Note that if multiple accordions exists on one page, the default id's of panels is incremented across the accordions, so it is recommended to override with custom id's.

To allow only one item to be open at a time, add [closeOthers]="true"

Styling

class="accordion-flush" is the main visually preferred variant for FHI.
